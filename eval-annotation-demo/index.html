标注规则说明

红框 = OCR 置信度低（L1）   黄框 = BERT 疑似错字（L2）
眉批：在段落前加 <眉批内容>。选中文本 → 点 <> 眉批 自动包裹并换行；未选中 → 光标处插入 <>。
小字（夹注）：用 【小字内容】 包裹。规则同上。
圈点符号（不区分颜色；有选区时逐字插入，无选区时光标处插入一个）：
  ● 黑色实心点   ○ 空心点（句点/表音/圈点都用它）
  、 顿号（不区分空心/实心）   ◎ 双圈
  -加-重- 横线加重（字间插入 -）
撤销：工具栏"↶ 撤销"或 Ctrl+Z，可撤销最近 50 步（跨页有效）。
自动保存（3 重保险）：① 改动自动存浏览器本地（刷新不丢）；② 点"📎 绑定 data.json"后实时写回磁盘上的 JSON（需 Chrome/Edge）；③ 随时可点"💾 导出 JSON 备份"或"下载 Excel"留档。
