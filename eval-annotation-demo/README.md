# 标注网站设计 Demo（SongPanda 2.3 · 眉批 & 圈点）

本文件夹是一个**可直接打开的标注工具前端 Demo**，用于向评委展示 SongPanda 2.3 在「古籍圈点 / 眉批」任务上的**标注网站设计**。

## 如何查看

直接双击打开 **`index.html`** 即可（推荐用 Chrome / Edge）。

- 左侧：书影缩略列表（本 Demo 精选 3 张代表性样本：含圈点、眉批的真实古籍书影）。
- 中间：原图预览，叠加 OCR 行框（蓝）与质检预警框（红 = OCR 低置信度 L1、黄 = BERT 疑似错字 L2）。
- 右侧：文本校勘编辑器 + 标注工具栏：
  - **结构**：`<>` 眉批、`【】` 小字（夹注）；
  - **圈点**：`●` 实心点、`○` 空心点、`、` 顿号、`◎` 双圈、`-加重-` 横线加重；
  - 支持逐字插入圈点、撤销（Ctrl+Z，可跨页）、按页「确认此页无误」、导出 Excel / JSON 留档。
- 右上角「📘 规则说明」可查看完整标注规范弹窗。

## 设计要点

- **双级质检预警**：把 OCR 置信度（L1）与 BERT 错字建议（L2）用红/黄框直观标在图上，辅助标注人员快速定位疑点。
- **圈点符号体系**：为古籍特有的「古人读写实践」符号（圈、点、顿、双圈、加重）设计了专用标注交互，可与正文、`【】` 小字、`<> `眉批共存于同一文本流。
- **可审计**：每一步修改支持本地自动保存、实时写回 `data.json`、导出 Excel/JSON 备份，保证标注过程可追溯。

## 数据来源说明

- 3 张样本图像均来自「全球汉籍影像开放集成系统」的**真实古籍书影**（圈点本 / 批校本），非合成数据。
- 本 Demo 仅截取少量页面用于展示界面与交互设计，完整评估集见项目 `评测集【共三期】/`。

> 注：双击以 `file://` 方式打开时，标注修改仅暂存于浏览器本地（刷新不丢）；如需实时写回文件，可通过本地静态服务器（如 `python3 -m http.server`）打开，再点「📎 绑定 data.json」。
